# 電線管占有率計算 ドメイン仕様書

## 概要

電線管（コンジット）に電線を収容する際、内線規程に基づいた占積率の計算と判定を行う。

## 用語定義

### 占積率（せんせきりつ）
正式な技術用語。電線の断面積合計を管の内断面積で割った値（%）。設計・内線規程の判断に使用。

### 占有率（せんゆうりつ）
現場用語。ケーブルがどれくらい「詰まっているか」の体感的な評価。施工性判断に使用。

本アプリでは「占積率」を正確に計算しつつ、一般的に馴染みのある「占有率」という表現も使用する。

---

## 計算式

### 占積率の計算

```
占積率(%) = 電線の断面積の合計 ÷ 管の内断面積 × 100
```

### 電線の断面積（外径から算出）

```
断面積(mm²) = π × (外径 / 2)² = π × 外径² / 4
```

### 管の内断面積

```
内断面積(mm²) = π × (内径 / 2)² = π × 内径² / 4
```

---

## 基準値

### 基本条件: 32%以下
電線の断面積合計が管内断面積の**32%以下**であること。これが標準基準。

### 例外条件: 48%以下
以下の条件を**すべて**満たす場合に限り、48%まで緩和可能：

1. **同一サイズの電線のみ**を収容（複合でない）
2. **管の屈曲が少ない**（直線に近い）
3. **引き入れ・引き抜きが容易**な構造

### 複合時のルール
異なるサイズの電線が混在する場合（複合）は、**無条件で32%を適用**する。

## 電線管の種類

### E管（薄鋼電線管）
- 正式名称: 薄鋼電線管
- 材質: 鋼
- 特徴: 一般的な屋内配線に使用

### G管（厚鋼電線管）
- 正式名称: 厚鋼電線管
- 材質: 鋼
- 特徴: 機械的強度が必要な箇所に使用

### PF管（合成樹脂製可とう電線管）
- 正式名称: 合成樹脂製可とう電線管
- 材質: 合成樹脂
- 特徴: 柔軟性があり、曲げ配管が容易

### VE管（硬質塩化ビニル電線管）
- 正式名称: 硬質塩化ビニル電線管
- 材質: 硬質塩化ビニル
- 特徴: 耐食性に優れる

## 電線の種類

### IV（ビニル絶縁電線）
- 正式名称: 600V ビニル絶縁電線
- 用途: 屋内配線（一般）
- 特徴: 単線または撚線

### VVF（ビニル絶縁ビニルシースケーブル）
- 正式名称: 600V ビニル絶縁ビニルシースケーブル平形
- 用途: 屋内配線（最も一般的）
- 特徴: 2〜4心、扁平形状のため「換算外径」を使用

### CV（架橋ポリエチレン絶縁ビニルシースケーブル）
- 正式名称: 600V 架橋ポリエチレン絶縁ビニルシースケーブル
- 用途: 電力配線
- 特徴: 耐熱性に優れる

### KIV（機器用電線）
- 正式名称: 600V 機器用単心ビニル絶縁電線
- 用途: 配電盤内配線
- 特徴: 可とう性に優れる

### EM-IE（エコ電線）
- 正式名称: 600V 耐燃性ポリエチレン絶縁電線
- 用途: 環境配慮が必要な屋内配線
- 特徴: ハロゲンフリー、燃焼時に有害ガスを発生しない

## 警告レベルの判定ロジック

```
if (occupancyRate <= 32) {
  return 'safe';      // 緑: 基準内（安全）
} else if (occupancyRate <= 48 && !hasMixedSizes) {
  return 'warning';   // 黄: 48%基準内だが注意
} else {
  return 'danger';    // 赤: 基準超過
}
```

---

## 48%適用を避けるべきケース

以下の条件に該当する場合は、48%基準ではなく32%基準を適用すべき：

- 点検口がない
- 将来的な引き替えが困難
- 曲げが多い（2回以上）
- ケーブルが太くて硬い
- 管が長い（10m以上）

**迷ったら32%にするのが安全**

---

## VVFケーブルの換算外径

VVFは扁平形状のため、断面積計算には「換算外径」を使用する。

換算外径の計算方法：
```
換算外径 = √(長辺 × 短辺)
```

または、製造元のカタログ値を使用する。
